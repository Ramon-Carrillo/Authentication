# Authentication


![screenshot](./img/screenshot.png)

## Built With

- HTML
- EJS
- Bootswitch
- Bootstrap
- NodeJS
- Mongoose
- Express
- MongoDB
- Passport

### Usage

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000
```

### MongoDB

Open "config/keys.js" and add your MongoDB URI, local or Atlas
